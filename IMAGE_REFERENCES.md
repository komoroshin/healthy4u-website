## Инструменты для создания изображений

### Для диаграм и схем:
- **Figma** (рекомендуется) - профессиональный UI/UX дизайн
- **Canva** - быстрое создание инфографики
- **Excalidraw** - hand-drawn style diagrams
- **Miro** - collaborative whiteboarding

### Для графиков и charts:
- **Chart.js** - JavaScript charts
- **D3.js** - data visualization
- **Google Charts** - simple charts
- **Tableau** - professional analytics

### Для скриншотов:
- **CleanShot X** (Mac) - annotated screenshots
- **ShareX** (Windows) - screen capture tool
- **Browser DevTools** - responsive design screenshots

## Размеры изображений

Все изображения используют:
- **Ширина**: 1200px
- **Высота**: 675px (aspect ratio 16:9)
- **Формат**: WEBP или JPEG (оптимизированный)
- **Качество**: 80-90% (баланс качество/размер)

Для Hero section:
- **Ширина**: 800px
- **Высота**: 1000px (vertical)
